# EF Core Migrations

VS Code extension to create Entity Framework Core migrations using a QuickPick/InputBox wizard.

## Development

1. `npm install`
2. Press `F5` in VS Code (`Run Extension`)
3. In the Extension Development Host, open the Command Palette and run `EF Core Migrations: Add Migration Wizard`

## Requirements

- `.NET SDK` installed (`dotnet`)
- `dotnet-ef` tool available
- A workspace containing one or more `.csproj` files with EF Core setup

## Current features

- Add migration wizard (`efcore.addMigrationWizard`)
